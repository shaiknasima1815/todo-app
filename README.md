# To-Do List App

## Overview

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript. This application helps users manage their daily tasks efficiently by allowing them to add, complete, and delete tasks.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Responsive and user-friendly interface
* Real-time task management

## Technologies Used

* HTML5
* CSS3
* JavaScript
